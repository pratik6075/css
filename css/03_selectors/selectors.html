what is selector?

  • a css selector is the first part of a css rule
  • it is a pattern of element and other teerm
  • the element or elements which are selected by the selector are refered to as the subject of the selector

types of selector are as below

  • universal selector
  • element selector
  • id selector
  • class selector
  • decendant selector
  • child selector
  • adjecent sibling selector
  • general sibling selector
  • group selectro
  • attribute selector
  • pseudo class selector
  • pseudo element selector
